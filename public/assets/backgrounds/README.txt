FONDO DE PANTALLA PERSONALIZADO (DINÁMICO)
--------------------------------------------
El sistema ahora detecta automáticamente en qué pantalla estás y busca un fondo específico.

REGLAS DE NOMBRE:
Añade tus imágenes (PNG/JPG) con el formato `bg-{nombre_pantalla}.png`.

- `bg-global.png`      -> Fondo por defecto (si no existe uno específico).
- `bg-dashboard.png`   -> Fondo específico para el Dashboard.
- `bg-profile.png`     -> Fondo específico para el Perfil.
- `bg-plants.png`      -> Fondo para la lista de plantas.
- `bg-diary.png`       -> Fondo para el diario.
etc...

Si no existe el específico (ej: `bg-diary.png`), la app usará automáticamente `bg-global.png`.
Si no existe ninguno, se verá el fondo original de la app.

Recomendaciones:
- Tamaño: Grande (ej: 1080x1920).
- Opacidad: No muy invasiva.
